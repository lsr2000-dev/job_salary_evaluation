# Changelog

本文件记录 **工作所得评估计算器** 的版本变更。格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [Unreleased]

### Added

- **UI**：顶栏「示范」按钮强化样式（`.btn-demo`）；导航栏下方新手指引条，提示点击「示范」载入范例，可点「知道了」关闭或进入示范后自动不再显示（`localStorage` 记忆）。
- **收入结构对比**：在对比表上方增加堆叠条，按比较货币展示每份工作税前毛收入中个人扣缴、必要开支与可支配所得的占比；汇率加载中显示占位；毛收入无效或汇率不可用时显示说明；可支配为负时以「超支」段表示缺口。

### Changed

- **收入结构图**：横轴改为比较货币下各列毛收入的**共享比例尺**（0～当前最高毛收入），条长反映绝对高低；加强可支配段默认样式；段上悬停缩放并显示含金额、占毛收入比与分项（税费来自个税明细，开支按住房/通勤/饮食/其他推导）的浮层。
- **收入结构图**：堆叠条改为 **SVG** 绘制（统一 viewBox 比例与横轴 Gmax），左缘与刻度对齐更稳定，便于后续扩展。

### Fixed

- **收入结构图**：绘图区改为与横轴同宽的 **0～Gmax** 单一坐标，段宽按金额/Gmax 映射，修复原先「条形容器再按 gross/Gmax 缩放」导致的跨行绝对值比例错误（如可支配更少却更长的观感）。
- **收入结构图**：移除段内 SVG 文字，避免非等比拉伸导致百分比被压扁；占比与金额以悬停浮层为准。
- **收入结构图**：横轴刻度（0 与最大值及分位）与中间**绘图列**同宽对齐，并增加基线与分位竖线，避免刻度相对整卡宽度错位。
- **收入结构图**：图例拖动排序改为标准「移动到下标」算法，**任意方向**拖动均可重排；增加图例区说明文案与拖放高亮。
- **收入结构图**：对比表输入（含同格连续输入跳过重绘表格时）会**同步刷新**收入结构图，无需整页刷新。

### Added

- **收入结构图**：顶部图例支持**拖动排序**，持久化至本地存档；排序决定每条堆叠条内三段**从左到右**的展示顺序，便于以左缘为基准阅读某组成部分的绝对尺度。
- **UI**：页面右下 **「↑ 图表」** 悬浮按钮，平滑滚动回收入结构图区域。

## [1.0.0] - 2026-04-08

初代正式发布：**多工作横向对比**、**税前税后与时间与效率**，以及 **中国大陆 / 美国 W-2 / 香港 / 不使用（含全球·远程）** 等税制与地区组合。

### Added

- **对比工作列**：最多同时对比 5 份工作；支持新增、删除、复制列；表头配置国家/地区、二级区划（中国大陆为省/市行政区划选择；美国为州份码 `subRegion`）、收入货币、税制。
- **收入与奖金**：税前基本工资、绩效工资、年终奖金系数、自定义额外税前收入；顶栏 **按月 / 按日(工作日) / 按年** 切换展示口径。
- **工作日模式**：国家法定工作日、大小周、单休等，用于推算工作日数与相关折算。
- **在岗与时间**：日均在岗时长、在岗休息；区分实际工作时长与含通勤时间；**报酬效率**（每分/秒、每小时，可选是否含通勤）。
- **中国大陆税制（估算）**：五险一金（可选）与个人综合所得个税估算；表头可选省/市，并关联 **社会平均工资** 等提示（数据来自 `data/cn-social-wage.json` 等）。
- **美国 W-2（估算）**：联邦所得税（Single + 标准扣除）、FICA（Social Security + Medicare，含高收入附加）、部分州个人所得税；表头选择州份（如 CA / TX 等，以页面选项为准）。
- **香港（估算）**：雇员强积金（MPF）与薪俸税（累进 vs 标准税率取低等简化逻辑）；婚姻/免税额相关选项。
- **不使用 / 待实装**：本列可不计算税费与法定扣缴；**全球 / 远程** 等地区可与「不使用」等组合用于对比占位；税制下拉含「待实装」选项。
- **开支**：住房租金、通勤成本与时间、工作日饮食（早午晚与零食）、自定义额外支出。
- **比较货币与汇率**：顶栏选择比较用统计货币；非本币收入按汇率折算展示（汇率来源以页面实现为准）。
- **界面与主题**：深色/明亮主题切换；填写区与计算区 **可折叠**，状态可随浏览器存档保留；顶栏 **填写项** 可在「全部」与「仅统计结果」间切换。
- **示范数据**：通过 **示范** 载入 `demo-state.json`（需 HTTP 访问，非 `file://`）；进入/退出示范与本地 `localStorage` 存档逻辑。
- **持久化**：主要输入与折叠状态保存在浏览器本地，刷新后可恢复。

### Changed

- 无更早版本；自本文件起建立变更记录。

### Notes

- 各税制均为 **简化估算**，页面与文档中的说明 **不构成税务或法律建议**。
- 若需自行托管（如 GitHub Pages），请使用 **HTTP/HTTPS** 打开，以便加载 `demo-state.json` 与 `data/` 下资源。
